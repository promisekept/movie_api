Business Logic URL HTTP Method Parameters Request body data format Response body data format
Return a list of ALL movies to the user /movies GET
Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user /movies/[title]
Return data about a genre (description) by name/title (e.g., “Thriller”) /genres/[NameOrTitle]
Return data about a director (bio, birth year, death year) by name /directors[Name]

Allow users to update their user info (username) /users/[Username]

Allow users to remove a movie from their list of favorites /users/[RemoveMovie]
